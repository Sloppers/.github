Maintainers are drowning in AI-generated contributions that look clean but add nothing. Slopper is an open source initiative to fight back — scoring every contribution using deterministic heuristics to answer one question: does this contribution actually add value?

### what we build

- [Slopper](https://github.com/Sloppers/Slopper) — GitHub Action that scores contributions and blocks low-quality contributions automatically, available on the [GitHub Marketplace](https://github.com/marketplace/actions/slopper)
